| --- | --- | --- |

| `workflow-definition-v1.0.json` | https://docs.comfy.org/specs/workflow_json | 2026-05-23 (same body as `comfy-router/schemas/nodes/workflow-definition-v1.0.json`) |

Refresh procedure:

1. Fetch the `.md` spec page from docs.comfy.org (or copy v1.0 from comfy-router after upstream sync).
2. Extract the fenced `json` block (strip a leading `theme={null}` line if present).
3. `json.loads` and rewrite the file with stable indentation.
4. Run `python scripts/validate_workflow_json.py workflows/` and fix any regressions.
