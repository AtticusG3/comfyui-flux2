## v1.3.0 - ERNIE-Image Pack and NVFP4 Community Override

### Highlights

- New `ernie-image` pack: Baidu's 8B Diffusion Transformer text-to-image with built-in 3B Prompt Enhancer.
- High VRAM tier loads the official Comfy-Org ERNIE-Image SFT (~16GB BF16, ~50 steps).
- Low VRAM tier loads the community ERNIE-Image-Turbo FP8 quant from `Abiray/ERNIE-Image-Turbo-FP8-NVFP4` (~8.2GB, 8-step distilled).
- Both tiers ship `ministral-3-3b` text encoder, `ernie-image-prompt-enhancer`, and `flux2-vae`.
- Bundled official Comfy-Org workflow templates: `ERNIE-Image - Text to Image.json` and `ERNIE-Image-Turbo - Text to Image.json` (Turbo workflow patched to load the FP8 quant from the Abiray repo).

### NVFP4 Community Override

Set the following to swap the Turbo FP8 quant for the smaller NVFP4 community quant (~4.8GB):

```
NVFP4_SUPPORTED=true
NVFP4_MODE=allow-community
LOW_VRAM=true
MODELS_DOWNLOAD=ernie-image
```

Both the model URL/filename and the bundled Turbo workflow are rewritten automatically.

### Other Changes

- Refreshed base Flux 2 Klein workflow templates to current ComfyUI shape.
- Replaced placeholder example prompts across bundled workflow nodes with curated Japan/JDM/anime themed prompts.
